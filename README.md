# Кошелёк кармы — Telegram бот

Бот для регистрации участников, начисления/списания “кармы” (баллов), квиза и сбора вопросов на открытый микрофон.

## Участнику (пользователю)

### Старт и регистрация

1. Открой чат с ботом и нажми `/start`.
2. Введи **номер с браслета** — ровно **4 цифры**.
3. Введи **имя** (как к тебе обращаться).
4. После приветствия нажми кнопку **«Да!»** (в сообщении бота) — это начислит **приветственные 100 баллов** (один раз).

### Главное меню (кнопки)

После регистрации снизу появляется клавиатура:

- **«Где взять карму»** — подсказки и быстрые переходы к активностям/квизу.
- **«Все активности»** — список активностей. Можно открыть карточку активности.
- **«Мой баланс»** — показать текущий баланс кармы.
- **«Списать карму»** — подсказка: при оплате мерча назови **ID браслета**.
- **«Квиз о банке»** — старт квиза прямо в чате.
- **«🎤 Задать вопрос (открытый микрофон)»** — отправить вопрос (лимит **3 вопроса** на человека).
- **«Меню»** — вернуться в главное меню и сбросить текущий сценарий ввода.

### Квиз о банке

1. Нажми **«Квиз о банке»**.
2. Нажми **«Стартуем!»** — бот задаст **один случайный вопрос**.
3. Выбери вариант ответа.

- За **правильный ответ** начисляется **10 баллов кармы** (награда **1 раз за квиз**, повторно не начисляется).
- После ответа бот показывает пояснение.

### Лекции (кодовые слова)

В **«Все активности»** открой нужную лекцию и нажми **«Ввести кодовое слово»**, затем отправь код одним сообщением.

Коды:

- **Лекция 1** — `коллеги`
- **Лекция 2** — `капучинка`
- **Лекция 3** — `вайбик`

Награда за лекцию: **400 баллов** (каждая лекция — **1 раз**).

## Админу (сотруднику стенда / магазина мерча)

Админка нужна для:
- начисления кармы за активности (кроме регистрации/квиза/лекций по промокоду),
- списания кармы при покупке мерча.

### Как получить доступ

1. В чате с ботом выполни команду `/myid` — бот покажет твой Telegram ID.
2. Добавь этот ID в переменную `ADMIN_IDS` в `.env` (через запятую).
3. Перезапусти бота.

### Вход в админ-меню

Команда: `/admin`

Если доступ есть — появятся кнопки:

- **«➕ Начислить карму»**
- **«➖ Списать карму»**
- **«🔄 Сброс админ-панели»** — прерывает текущий сценарий ввода (ID/суммы) и возвращает в админ-меню.

### Начисление кармы

1. Нажми **«➕ Начислить карму»**
2. Введи **ID браслета** участника (только цифры).
3. Выбери активность кнопкой в сообщении — начисление выполнится, бот покажет новый баланс.

Важно:
- Через админку **нельзя** начислять за **регистрацию** и **квиз** (это начисляется самим участником).
- Через админку **нельзя** начислять за **лекции** (они засчитываются по кодовому слову).
- Если лекцию не удалось засчитать кодом, в тексте админки есть подсказка про “Лекция (админ)” — это отдельная активность, **до 4 начислений** на одного человека.

### Списание кармы (магазин мерча)

1. Нажми **«➖ Списать карму»**
2. Введи **ID браслета** участника.
3. Введи сумму списания (целое число, например `50`).

Бот:
- уменьшит баланс,
- покажет новый баланс админу,
- попробует отправить участнику уведомление о списании.

## Запуск (для организаторов / разработчиков)

### Требования

- Python 3.10+ (рекомендуется)

### Установка зависимостей

В папке `tg_coin_bot`:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Настройка окружения

В `tg_coin_bot/.env`:

- **`BOT_TOKEN`** — токен Telegram-бота
- **`ADMIN_IDS`** — Telegram ID админов через запятую
- **`DB_PATH`** — путь к sqlite базе (по умолчанию `bot.db`)

Важно: **не коммитьте** реальные токены. Держите `.env` локально.

### Запуск

```bash
cd tg_coin_bot
python main.py
```

При первом запуске бот создаст sqlite-базу и таблицы автоматически.
